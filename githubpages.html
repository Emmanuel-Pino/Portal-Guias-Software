Cómo usar GitHub Pages para publicar tu primera página web

En esta guía aprenderás a publicar un sitio web de manera gratuita usando GitHub Pages, paso a paso.

  1. 1. Crear un repositorio

    Entra a GitHub, crea un nuevo repositorio público con el nombre mi-pagina y selecciona la rama main.

  2. 2. Subir archivos HTML

    Dentro del repositorio, haz clic en Add file → Upload files y sube tu archivo index.html. Luego confirma con Commit changes.

  3. 3. Activar GitHub Pages

    En la pestaña Settings, ve a la sección Pages. Selecciona la rama main y la carpeta root. Guarda los cambios.

  4. 4. Ver tu sitio online

    Después de unos segundos, tu web estará disponible en una URL, por ejemplo:

    https://tuusuario.github.io/pagina/

    LISTO Tu primera página web está publicada.
