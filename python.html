Cómo instalar Python en Windows

En esta guía aprenderás paso a paso cómo instalar Python en tu computadora con Windows y configurar un editor de código (IDE) para programar.

  1. 1. Descargar Python

    Visita el sitio oficial de Python en python.org/downloads y descarga la versión más reciente.

  2. 2. Ejecutar el instalador

    Haz doble clic en el archivo descargado. En la ventana del instalador, marca la casilla “Add Python to PATH” y luego haz clic en Install Now.

  3. 3. Verificar instalación

    Abre la terminal de Windows (cmd o PowerShell) y escribe:

    python --version

    Si ves la versión instalada, ¡listo! Python está funcionando.

  4. 4. Instalar un IDE (Editor de código)

    Para escribir y ejecutar tus programas de manera más cómoda, se recomienda instalar un IDE. Una de las mejores opciones es Visual Studio Code.
