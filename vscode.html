Cómo instalar Visual Studio Code

En esta guía aprenderás a instalar Visual Studio Code, el editor de código más usado por programadores.

  1. 1. Descargar VS Code

    Accede a la página oficial en code.visualstudio.com y descarga la versión para tu sistema operativo.

  2. 2. Instalar el programa

    Ejecuta el instalador, acepta los términos y condiciones, y selecciona las opciones recomendadas. Finalmente haz clic en Instalar.

  3. 3. Abrir VS Code

    Al finalizar la instalación, abre Visual Studio Code. Deberías ver la pantalla de bienvenida con las opciones de configuración inicial.

  4. 4. Instalar extensiones recomendadas

    En el menú lateral izquierdo, haz clic en el ícono de extensiones (cuatro cuadritos) y busca extensiones como:

    • Python
    • C#
    • Prettier
    • ESLint
