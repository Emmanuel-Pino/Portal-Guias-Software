Cómo instalar Node.js

En esta guía aprenderás a instalar Node.js en Windows para configurar tu entorno de desarrollo con JavaScript.

  1. 1. Descargar Node.js

    Ve al sitio oficial de Node.js en nodejs.org y descarga la versión LTS (Long Term Support), que es la más estable.

  2. 2. Instalar Node.js

    Ejecuta el instalador descargado. Acepta los términos y deja las opciones por defecto, luego haz clic en Instalar.

  3. 3. Verificar instalación

    Abre la terminal (cmd o PowerShell) y escribe:

    node -v

    Esto mostrará la versión de Node.js instalada.

    También puedes verificar npm (el gestor de paquetes incluido) con:

    npm -v

  4. 4. Probar Node.js

    Crea un archivo llamado app.js con el siguiente contenido:

    console.log("Hola desde Node.js");

    Ejecuta el archivo con:

    node app.js

    Si ves el mensaje en la terminal, Node.js está funcionando correctamente.
