Cómo crear y clonar un repositorio en GitHub desde Visual Studio Code

En esta guía aprenderás a crear un repositorio en GitHub, clonarlo en Visual Studio Code y subir tus primeros cambios.

  1. 1. Crear un repositorio en GitHub

    Entra a GitHub y haz clic en New Repository. Ponle un nombre (ejemplo: mi-proyecto), marca la opción Public y confirma con Create repository.

  2. 2. Clonar el repositorio en VS Code

    Abre Visual Studio Code, presiona Ctrl + Shift + P y escribe Git: Clone. Pega la URL de tu repositorio (ejemplo: https://github.com/tuusuario/mi-proyecto.git).

  3. 3. Conectar cambios y hacer commits

    En la barra lateral de VS Code, verás el ícono de Source Control. Haz tus cambios en el código, luego escribe un mensaje en la caja de commit (ejemplo: Primer cambio) y presiona Commit.

  4. 4. Subir cambios a GitHub

    Haz clic en Sync Changes o usa el menú desplegable para ejecutar git push. Esto enviará tus cambios al repositorio en GitHub.

    LISTO Ahora tu proyecto está sincronizado con GitHub.
